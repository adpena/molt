# llvm_boxed_inplace_dispatch_fact

## Summary

Moved LLVM boxed augmented-assignment dispatch off a local
`matches!(op.opcode, OpCode::InplaceAdd | OpCode::InplaceSub | OpCode::InplaceMul)`
hand-set.

The source fact now lives in `op_kinds.toml` as
`boxed_runtime_inplace_dispatch_opcodes` and renders to the exhaustive
`opcode_uses_boxed_runtime_inplace_dispatch_table`. LLVM lowering uses that
table for the first-class OpCode half while preserving the existing
`_original_kind.starts_with("inplace_")` namespace path for Copy-carried
in-place spellings.
