# Examples

This directory contains small, user-facing programs and focused reference workloads.

## Layout

- `hello.py`: minimal compile/run smoke example
- `edgebox/`: edge/runtime example with its own local README
- `cloudflare-demo/`: Cloudflare-oriented example assets
- `microgpt/`: larger model-oriented example workload
- `gpu_*.py`: GPU-oriented exploration examples

Use these examples for quick manual validation and documentation-friendly entrypoints.
