# Demo Guide

This directory holds vertical-slice demo applications and packaging-oriented sample apps.

## Canonical Entry Points

- Django offload quickstart: [docs/demo/DJANGO_OFFLOAD_QUICKSTART.md](../docs/demo/DJANGO_OFFLOAD_QUICKSTART.md)
- `django_app/`: demo Django application
- `molt_worker_app/`: worker-oriented demo package and export manifest

Treat `demo/` as integration examples, not as a catch-all artifact area.
